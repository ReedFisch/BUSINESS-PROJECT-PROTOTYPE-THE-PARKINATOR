---
description: Automatically commit changes after completing each user request
---

After completing any code changes or modifications requested by the user:

// turbo-all

1. Stage all changes:
```bash
git add -A
```

2. Commit with a descriptive message summarizing what was changed:
```bash
git commit -m "[descriptive message of changes made]"
```

3. Push to GitHub:
```bash
git push
```

**Note**: The commit message should briefly describe what was accomplished in the request.
